Towers of Hanoi

Denne gennemgang viser hvordan man kan bruge rekursion til at løse “Towers of Hanoi” spillet.

Her er python kode fra videoen.

def move(f,t):
    print("Flyt: {} ==> {}!".format(f,t))

def moveVia(f,v,t):
    move(f,v)
    move(v,t)

def hanoi(n,f,h,t):
    if n == 0:
        pass
    else:
        hanoi(n-1, f, t, h)
        move(f,t)
        hanoi(n-1, h, f ,t)

Her er en anden gennemgang af towers of hanoi.
