Selectors

Der er flere måde at udvælge elementer vha. selectors. De mest almindelige er:

  • Direkte ved hjælp af navn på elementet.
  • Ved brug af en klasse.
  • Ved brug af et ID.

Element

Man kan bruge html elementer direkte som selector, så vil alle elementer der matches bruge samme stil.
